Getting the Google AI Studio Key

To use the powerful AI Generator features, you need a free API key from Google AI Studio. This key acts as your personal access pass to Google's Gemini models.

  1. Navigate to Google AI Studio: Click the button below to visit the official Google AI developer portal.
  2. Sign In: Log in with your standard Google or Workspace account.
  3. Create API Key: Click on "Get API Key" in the left sidebar, and then click "Create API Key in new project".
  4. Copy your Key: Copy the generated string of letters and numbers.
  5. Save it in the Studio: Go back to AI Flowchart Studio, open the "Gemini API Settings" dropdown in the left panel, paste your key, and click "Save Key".

Privacy First

Your API key is stored 100% locally in your browser's LocalStorage. We never transmit it to any external databases or servers. It is used exclusively to make direct requests from your browser to Google's Gemini API.

✨ Using the AI Generator

The AI Generator translates your natural language descriptions into highly structured, professional flowcharts instantly.

  • Be Descriptive: The more detail you provide about the steps, the better the flowchart. Describe the start point, the decision forks, and the end points.
  • Wait for Generation: After clicking generate, the status pipeline will show you exactly what the AI is doing. It analyzes your prompt, generates the Mermaid.js structure, and validates it.
  • Example Prompt:
"Design a user authentication flow: user enters email → validate format. If invalid, show error and return to start. If valid, check if account exists. If not, prompt signup. If exists, send OTP → verify → grant access."

✏️ Using the Manual Builder

If you prefer full control, or if you want to modify an AI-generated flowchart, the Manual Builder is your go-to tool.

  • Adding Nodes: Provide a unique ID, a display label, and choose a geometric shape.
  • Adding Links: Connect two existing nodes by selecting the Source and Target. Optionally add a label to the link.
  • Editing Nodes: Click on any node directly inside the canvas to open the floating Node Editor. You can quickly change its label, shape, or delete it entirely.
  • Pan & Zoom: Use your mouse scroll to zoom in and out. Click and drag anywhere on the blank canvas to pan around.

Exporting your Diagram

Once your flowchart is perfect, use the top toolbar to export it in various formats.

  • Code: Copies the raw Mermaid.js code to your clipboard. Perfect for pasting into GitHub, Notion, or Markdown files.
  • SVG: Downloads a crisp, infinitely scalable vector graphic. Ideal for presentations or graphic design software.
  • PNG: Downloads a high-resolution raster image with a transparent background. Best for quick sharing in Slack or email.
