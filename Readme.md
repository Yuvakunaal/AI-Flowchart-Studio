# 🚀 AI Flowchart Studio

### Natural Language to Visual Logic | Multi-Agent AI Architecture

AI Flowchart Studio is a high-performance, enterprise-grade web application that transforms complex natural language descriptions into professional, structured flowcharts. Built with a sophisticated **Multi-Agent Orchestration** layer and a premium **Glassmorphic UI**, it offers a seamless experience for both AI-driven generation and manual precision building.

---

## ✨ Key Features

- **🤖 Multi-Agent AI Generator**: Powered by the Gemini AI models, our system uses an Orchestrator-Parser-Generator architecture to ensure logical accuracy and syntax validity.
- **🛠️ Precision Manual Builder**: A "Click & Type" interface for manual refinement, allowing you to seamlessly add nodes, link connections, and edit custom shapes with zero coding required.
- **💎 Ultra-HD Pro Export**: 
    - **PNG**: 3x Super-Scaled, crystal-clear captures using native SVG geometry. Direct download straight from your browser.
    - **SVG**: Scalable vector format for high-end print, presentations, and graphic design.
    - **Code**: Direct access to Mermaid.js syntax for version control and documentation.
- **🔍 Smart Zoom Engine**: Hardware-accelerated scaling (+/- 300%) with "Sticky Controls" and "Top-Origin" stability, keeping your diagram perfectly centered.
- **🎭 Intelligent Themes**: Dynamic switching between Dark, Light, Forest, and Neutral aesthetic themes.
- **🌍 Layout Versatility**: Instant toggle between Vertical (Top-Down) and Horizontal (Left-to-Right) flowchart orientations.
- **🔐 Privacy First**: Complete local-only execution. Supply your own Google AI Studio API Key which is stored securely in your browser's local storage and never sent to our servers.

---

## 🏗️ The Multi-Agent Engine

The core of the system is a sophisticated 4-stage pipeline that ensures "Garbage In" never becomes "Garbage Out". By splitting the workload across multiple specialized agents, the application achieves a near 100% syntactical success rate:

1.  **Orchestrator Agent**: Analyzes user intent and validates if the prompt can logically be visualized as a flowchart.
2.  **Logic Parser Agent**: Decomposes the raw text into a highly structured JSON graph containing strict Nodes and Edges.
3.  **Generator Agent**: Converts the validated logical graph into highly optimized Mermaid.js syntax.
4.  **Syntax Validator**: Performs a final safety check to catch rendering anomalies and guarantee a perfect visual output every time.

## 🛠️ Technical Architecture

- **Frontend**: ES6+ JavaScript, Native DOM APIs, Mermaid.js, html2canvas.
- **Backend**: FastAPI (Python), Google GenAI SDK (Gemini 3.1 Flash).
- **Styling**: Vanilla CSS3 with Custom Properties (Glassmorphism design system).
- **Architecture**: RESTful API with Server-Sent Events (SSE) for real-time pipeline status updates directly to the UI.
